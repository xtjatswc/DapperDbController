# DapperDbController

#### Description
实现了一个轻量级的数据库查询 API 工具，支持 SQL Server、MySQL、PostgreSQL 和 Oracle 四种主流数据库，可扩展。通过简洁的设计和灵活的架构，具备开箱即用、易于集成的特点。即使不使用 Dapper，只要是支持IDbConnection的 ORM 框架，也能轻松替换核心数据库操作部分。同时，该工具内置 Basic Auth 认证机制，保障 API 访问安全。
